## 7. Playbooks

### 7.1 Market Crash Playbook
If BTC drops >20% in 24 hours:
1. **Pause:** Stop all new buys.
2. **Assess:** Check if fundamentals changed (regulatory news, hack, etc.).
3. **DCA:** If no fundamental issue, deploy stablecoins to buy BTC/ETH at discount.
4. **Rebalance:** If allocation drifts >15%, rebalance to target.

### 7.2 Protocol Exploit Playbook
If a protocol is exploited:
1. **Immediate:** Withdraw all funds from protocol.
2. **Assess:** Determine if funds are recoverable.
3. **Log:** Record loss in portfolio tracker.
4. **Report:** Notify Finance Head within 1 hour.
5. **Blacklist:** Add protocol to "Do Not Use" list.

### 7.3 Profit-Taking Playbook
If portfolio gains >20%:
1. **Calculate:** 50% of profits → withdraw to fiat.
2. **Rebalance:** 50% of profits → compound into yield farming.
3. **Tax:** Log all transactions for tax reporting.
4. **Report:** Send monthly P&L summary to Finance Head.
