## 6. Execution Workflow

### 6.1 Sales & Onboarding
1. **Lead Source:** Client comes from Lead Qual cold email or standalone funnel.
2. **Intake Form:** Client fills form (industry, location, services, contact info).
3. **Payment:** Client pays via Stripe (auto-redirect to confirmation).
4. **Kickoff:** OpenClaw auto-generates site (no human call needed).

### 6.2 Site Generation (Automated)
1. **Template Selection:** Match industry to template (e.g., "Plumber" → Template #1).
2. **Copy Generation:** AI writes content (Home, Services, Contact) using industry keywords.
3. **SEO Optimization:** Auto-generate meta tags, local keywords.
4. **Form Integration:** Embed contact form → Lead Qual scoring → auto-followup.
5. **Deploy:** Push to Netlify/Vercel via API.
6. **Preview:** Send client link + "Approve or 1 revision" email.

### 6.3 Revision & Launch
1. **Client Feedback:** Client replies with revision request (max 1).
2. **Auto-Update:** AI adjusts copy/layout based on feedback.
3. **Final Approval:** Client confirms → Site goes live.
4. **Onboarding:** Send welcome email with Lead Qual dashboard link.

## 7. Playbooks

### 7.1 High-Value Client Playbook
If client requests "Growth Partner" tier ($997 + $297/mo):
1. **Priority:** Fast-track deployment (12 hours instead of 24).
2. **Personalization:** Add custom logo (if provided) + brand colors.
3. **Lead Qual Setup:** Pre-configure scoring rules for their industry.
4. **Follow-Up:** Schedule automated monthly performance report.
5. **Upsell:** Offer SEO or ads management after 30 days.

### 7.2 Difficult Client Playbook
If client demands endless revisions:
1. **Remind:** Politely reference "1 revision only" policy from intake.
2. **Offer:** One final "goodwill" revision (AI-generated).
3. **Escalate:** If still unsatisfied, offer refund (no hard feelings).
4. **Log:** Record reason for refund (improve template/intake process).

### 7.3 Template Gap Playbook
If client industry has no template:
1. **Check:** Is there a similar template? (e.g., "Electrician" → "Plumber").
2. **Adapt:** Use generic template + AI customize copy.
3. **Log:** Track demand for new template (if 5+ requests, create new one).
4. **Communicate:** Tell client "Customizing for your industry" (manages expectations).
