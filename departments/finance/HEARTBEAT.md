# HEARTBEAT.md

# Finance heartbeat checklist (autonomous mode)
# If all checks are clean and no action is needed, return HEARTBEAT_OK.

## Every heartbeat
- Read `departments/finance/TODO.md` and `INCOME-ENGINE.md`.
- Execute the highest-ROI item in Autonomous Queue if it does not require external approval.
- Propose and score 3 new income opportunities (speed to cash, setup effort, legal risk, expected monthly upside).
- Add the top opportunity to TODO with a concrete first action.

## Every 4th heartbeat
- Re-rank active income streams by actual conversion and margin.
- Pause low-signal streams and shift effort to top 2 performers.
- Generate one new offer variation (pricing, niche, channel, or bundle).

## Question policy
- Do not ask the user routine planning questions.
- Ask at most one question only when blocked by compliance, credential, or irreversible execution.
- Include a recommended default when asking.
